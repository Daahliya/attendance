# Student Attendance System

A simple and user-friendly Student Attendance System built using Python and HTML, designed to manage and record student attendance through a clean web interface.

## Features
- Simple and intuitive user interface
- Mark and manage student attendance
- HTML-based frontend for user interaction
- Python backend for processing attendance data
- Lightweight and beginner-friendly project

## Technologies Used
- Python
- HTML
-Flask

## Key Learning Outcomes
- Building web interfaces using HTML
- Connecting frontend forms with Python backend logic
- Handling user input and basic data processing
- Understanding simple web application flow
- Using GitHub for version control
